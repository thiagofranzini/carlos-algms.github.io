--- layout: post title: "Install Oracle java on Ubuntu" date: 2014-11-20T15:30:00-02:00 categories: Linux Ubuntu Java --- Ubuntu comes with an opensource java installed. To use the oracle's java, you need to remove the openjkd and install oracle's.

First we need to remove the openjdk from Ubuntu. 

sudo apt-get purge openjdk*

Then add webupd8team repository, since Oracle still don't have an official repository. 

sudo add-apt-repository ppa:webupd8team/java
sudo apt-get update

After the update finished, we can install java via: 

sudo apt-get install oracle-java8-installer
