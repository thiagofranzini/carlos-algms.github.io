Configure Ubuntu Server to Support Pt-Br

Every cloud that you use outside Brazil, does not come with pt-br support installed. When you access your server via SSH, almost every command that you run on terminal, will generate a warn that your language is not supported. To avoid this annoying warning, you just need to install the language pack.

Note: This will not translate your Ubuntu, this will only add support to your language and remove the errors. 

sudo apt-get install language-pack-pt-base
sudo dpkg-reconfigure locales

Configure timezone

After install the language-pack, is a good pratice configure the clock of your Ubuntu to reflect the clock of your location. 


sudo dpkg-reconfigure tzdata
